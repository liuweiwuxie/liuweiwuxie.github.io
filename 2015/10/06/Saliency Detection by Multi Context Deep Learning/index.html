Saliency Detection by Multi-Context Deep Learning

  • CNN
  • Global context
  • Saliency Detection
  • Superpixel
  • local context

CVPR 2015, Tang Xiaoou’s group, employing deep Convolutional Neural Networks to model saliency of objects in images. Global context and local context are both taken into account, and are jointly modeled in a unified multi-context deep learning framework.

Abstract

  1. low-level saliency cues cannot work very well with a low-contrast background
  2. Multi-context CNN: global context and local context
  3. Works:
    1. investigate different pre-training strategies + design a task-specific pre-training scheme
    2. investigate different models in ImageNet

Introduction

  1. Application of saliency detection
  2. 传统的saliency detection的方法的两个不足：
    1. 与背景对比度低
    2. 对于图像需要一些高级别的认知和理解
  3. CNN是怎么怎么的成功
  4. 局部的context很有利于局部物体的提取，但是显著性检测的话如果仅仅利用局部的内容的话就和物体检测没有什么区别了，所以需要运用到global context
  5. 贡献：
    1. global context + local context的一个CNN的模型；
    2. pre-training的策略以及比较了不同的模型

Related work

  1. Salient object segmentation
  2. Objectness and object proposal
  3. Hierarchical structure for saliency detection
  4. DCNN

Our Approach

  1. Global-context Modeling by Deep CNN:Superpixel-centered large context window
  2. Integrated multi-context model:
    1 公式推导没有细看
    2 怎么样将global context与local context结合起来
  3. Task-specific Pre-training: 具体内容就是介绍了四种pre-training的方法，以及具体superpixel在方法中是怎么使用的

Experiment

  1. Bechmark datasets
  2. Multi-context VS simgle context
  3. Comparison of different pre-training strategies
  4. Comparison of different CNN structures
  5. Comparison with other methods
