Is object localization for free

  • CNN
  • Object detection
  • Object recognition

I think this paper is very interesting! This paper focuse on the relationship between object classification and localization.

Abstract

  1. Image-level annotation VS object-level annotation
  2. A weakly supervised CNN: rely on image-level labels but can output location
  3. Accruate image-level labels, approximate locations, have performance comparably to its fully-supervised counterparts

Introduction

  1. Visual object recognition:
    1. whether the image contains instances of certain object
    2. location
    3. pose
    4. deformable object has a collection of parts
    5. each object may be cropped or patially occluded
  2. Three methods for object recognition:
    1. local image features->bag of visual words->classifier
    2. deformable part models
    3. CNN
  3. For CNN, object-level annotation is very good but not easy to get
  4. So the author thinks he can do:
    1. train CNN with image-level annotation, without location information, 这个问题的难点在于物体可能在不同的位置，不同的尺度，不同的观察点，并且两种经常出现在一起的物体怎么去区分！
    2. Modifications of CNN: last fully connected layers -> convolutional layers, max-pooling layer at the final, new cost function to learn from image-level supervision
  5. The modified CNN can output image-level labels, also can loalize.

Related work

  1. visual recognition最基本的挑战在于怎样对intra-class的appearance和shape进行建模
  2. what is weakly supervised learning?
  3. target output in visual recognition: image-level labels, location and bounding boxes of objects, object segmentation, 3D pose and geometry of objects
  4. 其他用CNN和image-level labels的方法并没有探究classification对于location prediction的影响。
  5. Contribution:
    1. end-to-end learning pipeline that learns from complex cluttered scenes containing multiple objects by explicitly searching over possible object locations and scales in the image.
    2. 不光image-level label预测准确，还可以准确地预测大致的位置，并且效果与fully-supervised的方法相比差不多甚至还要好

Network architecture for weakly supervised learning

Three modifications:

Convolutional adaptation layers

  1. Fully connected layers —> convolutional layers
  2. 我觉得主要目的是要使原先最后一个fully connected层输出一个每个位置的预测的结果（对于任意尺寸的输入图片），这样来得到一个receptive window里面物体种类的分数，相应的也就有了这个物体的位置信息

Global max-pooling layer at the output

  1. 使从上面一个modification出来的每个位置预测的结果变成一个一维的向量，有利于loss的计算

Multi-label classification loss function

  1. Treating a multi-label classification problem as K indepen-dent classification problems
