Image Super Resolution Using Deep Convolutional Networks

  • CNN
  • Super Resolution

The work of He Kaiming. TPAMI. I think there is a lot of work can be done in this aspect.

Abstract

  1. An end-to-end mapping between the low/high-resolution images(有点粗暴啊！)
  2. Spare-coding-based SR(super resolution) method -> a deep NN
  3. Different structures and parameter settings for fast speed and performance
  4. Color image

Introduction

  1. 简单介绍了下SR这样一个问题的一些特点: undetermined inverse problem, domain specific tasks, etc.
  2. the pipeline of the sparse-coding-based method
  3. SRCNN:
    1. good performance and fast speed
    2. larger datasets or models can boost the performance of SRCNN
    3. end-to-end
    4. build up the relationship between DNN-based SR method and traditional sparse-coding-based SR method
  4. Explain the extension of this work compared to the paper published in ECCV2014
    1. larger model
    2. one channel -> three channels
    3. new analyses and explainations
    4. extend the experiment parts

Related Work

  1. Image super-resolution: example-based methods are popular
  2. CNN
  3. DL for image restoration

Convolutional Neural Networks for Super-resolution

  1. 对所要研究问题的一个基本设定：
    1. Patch extraction and representation
    2. Non-linear mapping
    3. Reconstructuion
  2. 实际上面的三个方面都是用卷积层来实现的
  3. 将sparse-coding-based methods变成一个DNN来看待分析，试图说明两个方面的东西：
    1. SRCNN是end-to-end
    2. 对于SRCNN的hyperparameter的选取有着指导意义
  4. Training过程中loss function的选取, 一些细节的设定

Experiments

实验部分太多了，具体的conclusion都在每部分实验比较结束，TPAMI的实验就是要从各个方面去support你的hypothesis

Conclusion：

  1. 对于CNN来说，没有什么大的创新点，没有用到pooling layers，所以网络的结构不会非常的深，到底是什么样的feature可以使得对SR的恢复会比较好？Hypercolumn可以用来提feature吗？
  2. 对于SR问题来说，一个end-to-end的系统，尝试用DL来解决这样的问题是新的
  3. 对于SR问题的解决的popular pipeline: 从low resolution的patch中提取feature来恢复high resolution的patch
  4. whether deconvolutional layer will work？
