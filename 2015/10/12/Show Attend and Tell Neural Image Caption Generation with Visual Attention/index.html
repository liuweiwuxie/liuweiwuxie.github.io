Show Attend and Tell Neural Image Caption Generation with Visual Attention

  • Attention Model
  • CNN
  • Caption Generation
  • LSTM

The work of Bengio. I think this kind of attention model can be used in scene text detection and recognition.

Abstract

  1. An attention based model: 可以自动地学习描述图片的内容
  2. Maximizing a variational lower bound
  3. Visualize how the model learn to fix its gaze on salient objects and generate the corresponding words in the output sequence.
  4. Three benchmark datasets: Flickr8k, Flickr30k, MS COCO

Introduction

  1. salient visual information -> descriptive language: scene understanding
  2. cnn + rnn: the new methodology
  3. attention can help!
  4. models(a soft attention mechanism and a hard attention mechanism) + visualize what the model sees + validate on three datasets

Related work

  1. encoder-deconder framework
  2. Neural networks for caption generation (popular)
  3. object detector + RNN(LSTM), go beyond objectness

Image Caption Generation with Attention Mechanism

  1. Encoder: cnn features
  2. Decoder: LSTM
    1. how to get context vector: attention model
    2. 用一个权重来给予图片中不同物体一个attention的权重，所以每个时间t时的context vector实际上是图片中物体feature的一个加权和，权重是学习得到的

Learning Stochastic “Hard” vs Deterministic “Soft” Attention

  1. Hard: sample the attention location each time, 每次图片中选取一个物体来作为attention, 剩下的其实就是怎样用数学来把这样的一个东西用数学模型抽象出来
  2. Soft
  3. Doubly stochastic attention：不仅仅要在每个时间段上对attention的权重进行调整，还要在整个时间段上对attention的权重进行调整
  4. Training procedure：进行训练时候的一些操作

Experiments

  1. Evaluate only on a single model (ensembling model)
  2. Learning to attend

Contribution of this paper

  1. 我觉得最大的contribution应该是提出了attention model这样的一个东西到caption generation里面，并且提出了两个attention model，一个是hard的attention model，一个是soft的attention model，并且都用数学的形式把这个给抽象了出来，我觉得这个很可以用到文字的detection and recognition上面来。
