Efficient Object Localization Using Convolutional Networks

  • CNN
  • Coarse-to-fine
  • Multi-resolution
  • Pose estimation
  • SpatialDropout

CVPR 2015 Yann LeCun
This paper is using CNN to solve the problem of keypoint localization. The framework is a coarse-to-fine CNN. In my opinion, the contribution of this work :

  1. coarse-to-fine CNN framework
  2. spatialDropout
  3. The performance of this method for keypoint localization is very good.

We should carefully think about what makes CNN have a poor performance towards localization. For localization of different tasks, what is the difficult problem? like for scene text detection.

Abstract

1. Pooling layers in the CNN reduce localization accuracy.
2. Human joint location estimation
3. Cascade

Introduction

Advantages of pooling layers:

  1. prevent over-training
  2. reduce computational
  3. invariance to local input transformation
    Disadvantage of pooling layers:
  4. limit spatial localization accuracy

Related Work

1. DPM
2. CNN: mapping from input RGB image to XY location adds unnecessary learning complexity which weakens generalization
3. Multi-resolution CNN: Joint Training of a Convolutional Network and a Graphical Model for Human Pose Estimation

Methods:

Coarse Heat-Map Regression Model

Fine Heat-Map Regression Model



Before “+”, all the joints share the same parameters, but the last conv layer “128x36x36” is joint-specific.

Some details

  1. SpatialDropout: one whole channel of the feature map is dropped out instead of some positions
  2. Data augmentation: rotation and scale
  3. Joint training:
    1. Coarse heat-map model
    2. Hold the parameters of coarse heat-map model, train the fine heat-map model
    3. Train both: E = E1 + a x E2, a = 0.1

      Experiments

      Dataset: FLIC, MPII-Human-Pose
  4. Compare the impact of pooling on spatial precision
  5. Compare the performace of cascade model with coarse model only
  6. Compare the SpatialDropout with standard dropout
  7. Metion about scale invariance

I think this work is really good for keypoint detection. But the corse-to-fine strategy can be applied to other probelms.
