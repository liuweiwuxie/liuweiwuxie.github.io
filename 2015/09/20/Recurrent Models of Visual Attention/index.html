Recurrent Models of Visual Attention

  • Attention Model
  • RNN
  • dynamic visual control
  • object classification

Google DeepMind Attention Model

Abstract

Drawbacks of CNN: For detection, the complexity of computation increases linearly with the number of image pixels.
Model proposed: RNN select a sequence of regions and only process the selected regions at high resolution.
Characteristics of proposed model: translation invariance, computation is independent of the size of the image, non-differentiable, reinforcement learning

Introduction

Why we should use attention Model

  1. reduce the computation consumption
  2. focus on the most interesting part to abstract the most correct feature
    Location of Human attention should be task specific.

Previous Work

Three methods that have been applied to vision:

  1. Slide window: exclude useless slide windows as much as possible (No context information for object detection)
  2. Saliency detectors: low level image properties.
  3. a sequential decision task.

The Recurrent Attention Model (RAM)

Problem Setting: the attention problem as the sequential decision process of a goal-directed agent interacting with a visual environment.

Model:

  1. Sensor: retina-like representation, limited bandwidth
  2. Internal state: like hidden layer in LSTM
  3. Actions: sensor control and an environment action
  4. Reward: after an action, the model get a new visual observation and a reward signal r
  5. Training: i am not sure
  6. Variance Reduction
  7. Using a Hybrid Supervised Loss

Experiments

Settings of the Attention Model
MNIST image classification
Playing a simple computer games

Advantages of RAM

  1. Focus only on a small part of the image. More efficient
  2. Be able to handle different scales of the image\
  3. Learn without ground truth

Drawbacks of Recurrent Attention Model (RAM)

  1. The number of target is limited. 如果有多个目标，可以把多个目标都可以检测到吗？
  2. Applying this kind of method to large scale object recognition and video is still unrealistic. (If I can use R-CNN + RAM)
  3. retina encoding: if we can adopt another strategy to make the performance better?
  4. When to stop glimpsing when it has enough information to make a confident classification. I think the change should be made in the reward part.

My own opinion

  1. I think the choices related to the action function and the location are very important
  2. I don’t think for now this method can be used for large scale word detection and recognition.
  3. I am not familiar with the reinforcement learning, so some parts of this article I cannot understand. And about the whole training procedure, I am not very sure.
