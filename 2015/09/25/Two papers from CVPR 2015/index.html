Two papers from CVPR 2015

  • Bottom-up
  • CNN
  • Eye fixation prediction
  • Fine-grained image classification
  • Multiresoluiton CNN
  • Top-down
  1. The Application of Two-level Attention Models in Deep Convolutional Neural Network for Fine-grained Image Classification
  2. Predicting Eye Fixations using Convolutional Neural Networks

Part One

Problem

The whole process of fine-grained image classification (for an image):

  1. First of all, where is the object in an image?(Objection Detection)
  2. After we locate the object in an image, then we should know what the object belongs to. A dog or a cat?
  3. After we know the category of the object, then we should do the fine-grained classification.

Difficulty

For this kind of problem, what is the difficulty?

  1. The location: if we cannot locate the object in an image accurately, then it is hard to get the most suitable features for later steps.
  2. Inter-class & intra-class: we may use the whole features of an object to distinguish a dog from a cat (in most suituation). But when we try to separate different kinds of dog, features we may use are part-level or more detailed features of that object. How can we teach convolutional neural network to focus on different features that we want? Just from the training dataset? Or in other words, how can we design CNN or change the form of dataset fed into CNN to make it understand what we want?
  3. Intra-class: how to model all the kinds of objects in one category? How to encode the part information of each object in CNN? (Is it possible to use this paper “DeepID-Net: Deformable Deep Convolutional Neural Networks for Object Detection”)

Methods in this paper

  1. Use selective search to generate patches from input images (multi-scale & multi-view)
  2. Object proposal selection: use FilterNet to select patch images
  3. Use selected patches to train DomainNet
  4. Use hidden layers’ filters of DomainNet, perform spectral clustering to group the filters. Then we can get a representative part patch. (I doubt this step)
  5. Use selected part patched and original imageto build a SVM

Personal thinking

  1. The input patch images of 4 must contain all three the parts of an object. And directly resize the input patch image to the size of receptive field of filter is not wise. Because some patch image may contain the whole part of the object.
  2. 这篇文章一直claim的点就是他们没有使用object bounding和part-level的annotation，使用的是image-level的annotation.但是实际上selective search+FilterNet已经生成了object-level的bounding box和part-level级别的annotation然后使用DomainNet里面的filter去寻找part-level的annotation.

Part Two

Bottom-up visual saliency can also be inferred via combining information over multiple resolutions.

Problem: eye fixations

  1. Three steps of traditional approaches: early feature extraction, feature contrast inference and contrast integration
  2. This paper states that we should combine bottom-up saliency information and top-down factors to build a model for predicting eye fixations.

Contribution of this paper:

  1. Simultaneously follow the three steps of traditional approaches in the framework of CNN
  2. 效果好

Methods in the paper

  1. 10-fold cross-validation

This paper is quite simple.
